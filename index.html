FAQ

Можно ли использовать сайт как «App Store Landing Page»?

Да. Это одностраничник с быстрым CTA, SEO-мета и ссылками на политику/поддержку.

Нужен ли сервер?

Нет. Это статический сайт: GitHub Pages / Cloudflare Pages / Netlify / Vercel.

Как заменить контент под мой билд?

В файле index.html замените APP_STORE_LINK, тексты и блоки возможностей; скриншоты — в assets/screens.

Можно добавить форму обратной связи без бэкенда?

Да: например, Formspree/Netlify Forms (или собственный Firebase Function) — подключается за 5–10 минут.
